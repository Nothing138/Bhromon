🌍 Bhromon - Real-time Travel Networking App

**Bhromon** is a modern social platform for travelers to share their experiences and find group companions for their next adventure. Built with **Flutter** and **Supabase**, this app focuses on real-time interactions and seamless media sharing.

✨ Key Features
* **Dynamic Travel Feed:** Real-time post updates with images using Supabase Storage.
* **Privacy First:** Option to post anonymously to keep your identity private.
* **Group Sync:** Dedicated toggle for travelers looking to build or join a tour group.
* **Cross-Platform:** Fully compatible with Android, iOS, and Flutter Web.
* **Secure Backend:** Robust Row-Level Security (RLS) policies to protect user data.

🛠️ Tech Stack
* **Frontend:** Flutter (Dart)
* **Backend:** Supabase (PostgreSQL, Auth, Storage)
* **State Management:** StatefulWidget (Scalable for future integration)

🚀 How to Run
1. Clone the repository.
2. Run `flutter pub get`.
3. Set up your Supabase credentials in `main.dart`.
4. Run `flutter run -d chrome` (for web) or on your mobile device.
